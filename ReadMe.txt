■１．说明
	这是一个AVG+STG的简易游戏引擎，基于VB .NET和DxLib。
	目前BGM部分仍然未完工，存档仍未完工。
	在根目录下：
		DLLs为核心的DLL文件目录
		DATAs为资源目录
			SAVEs为存档目录，有一个范例
			SCRIPTs为脚本目录，有一个范例
			Config.txt为游戏配置文件
	脚本格式类似Ruby，可以使用相应的编辑器使用语法高亮
	（内置的开发工具仍然在开发中）


■２．安装方式
	借用神主的一句话，「既然是微软不支持的东西我们也不能支持」
	因此我用了.NET Framework 4.5
	——————————————————————————————————
	因为重制版更换了GUI因此这是必要的，遗憾的是无法对XP支持了
	请自行安装下.NET Framework 4.5
   

■３．卸载方式
